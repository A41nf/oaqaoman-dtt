# DTT Dashboard

## Description
A dashboard application for DTT with multiple pages including meetings and other features.

## Files
- `index.html` - Main dashboard page
- `meetings.html` - Meetings page

## Getting Started
Open `index.html` in your web browser to view the dashboard.

## Features
- Dashboard overview
- Meetings management
- Additional features available through navigation

## Usage
1. Open `index.html` in a web browser
2. Navigate through different sections using the available links
3. Visit `meetings.html` for meeting-related information
